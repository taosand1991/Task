  10. Prompt: Please explain - to a non-technical person - the difference between server side and client side rendering.

    Answer:

    Server side rendering can be executed by web servers and it is being used to create dynamic pages and it has access to the file system residing at the web servers. It can store data directly to the database and it does not run on web browsers. In other words, server side rending is a code that runs at the background that is not visible to the user. User can not interact with the code. It is used to create and generate dynamic pages.

    Client side rendering can be executed by the web browsers. It is run on the user's computer and the user will be able to interact with the page.In client-side rendering, user does not have access to the file system or database. It does not provide security for the data as well.
